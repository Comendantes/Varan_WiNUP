# Varan_WiNUP

> Утилита для быстрой и правильной настройки Windows от **Esarev**

![Version](https://img.shields.io/badge/version-1.0.3%20VaRaN-orange)
![Platform](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Что это?

**Varan_WiNUP** — десктопное Electron-приложение для настройки Windows. Всё что нужно после чистой установки — в одном месте.

## Возможности

- 🔧 **Системные твики** — тёмная тема, скрытые файлы, расширения, телеметрия, Edge
- 📦 **Установка программ** — Chrome, Discord, Telegram, Steam, 7-Zip, Notepad++ и другие
- 🛡️ **Обход блокировок** — Zapret, tg-ws-proxy для Telegram на ПК
- 💻 **Информация о железе** — CPU, GPU, RAM, диски
- 🎨 **Эстетика** — кастомные курсоры, советы по оформлению рабочего стола
- 💿 **Переустановка Windows** — пошаговые инструкции с Rufus
- ✅ **Активация Windows и Office** — через MAS (Microsoft Activation Scripts)
- 🔍 **Глобальный поиск** по всем разделам

## Стек

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Готовый установщик появится в папке `release/` как `Varan_WiNUP-Setup.exe`.
