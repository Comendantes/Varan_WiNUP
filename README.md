# VARAN_WiNUP

> Утилита для настройки и обслуживания Windows — красиво, быстро, по делу.

![Version](https://img.shields.io/badge/version-1.0.3%20VaRaN-brown)
![Platform](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## Что умеет приложение

| Раздел | Описание |
|---|---|
| 📋 Информация | Сведения о системе и горячие клавиши |
| 💽 Разделение диска | Управление разделами накопителя |
| 🪟 Windows и Office (MAS) | Активация Windows и Office без ключа |
| ⚙️ Системные твики | Полезные твики реестра и системы |
| 📦 Установка программ | Быстрая установка нужного софта |
| 🔓 Обход блокировок | Инструменты для обхода ограничений |
| 🖥️ Железо и драйверы | Сканирование железа и установка драйверов |
| 🎨 Эстетика и порядок | Кастомизация курсоров и внешнего вида |
| 🔄 Переустановка Windows | Пошаговая помощь при переустановке |
| 🔍 Глобальный поиск | Поиск по всем разделам приложения |
